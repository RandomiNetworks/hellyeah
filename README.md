HellYEAH
============

This is our first publicly available web application, powered by nodejs and express.

Requirements
-------

- Windows / OS X / Linux
- NodeJS


Installation
-------

Yay! Here are the instructions for installing the software.

    git clone https://github.com/RandomiNetworks/hellyeah.git
    npm install
    node app.js


Usage
-------

Start

    node app.js
